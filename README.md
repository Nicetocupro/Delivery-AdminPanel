# Delivery-AdminPanel

## 说明

本仓库是管理员面板(面向Web Browser)的上游仓库，马恒超作为Maintainer。

## 运行说明

本项目使用**vue3+Primevue组件库+PrimeIcons图标库**

'git clone'项目后输入如下命令运行项目。
```
# npm运行过慢可配置Taobao镜像cnpm
npm run install
npm run dev
```
